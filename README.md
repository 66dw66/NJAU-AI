# 南京农业大学 智慧农业学院（人工智能学院） 人工智能专业相关课程资料
## 介绍
##### 由于本人受到了很多学长学姐们的帮助，所以希望将这份帮助传递下去，让学弟学妹们能够有更多的时间放在提升自己上，而不是为了收资料而焦头烂额。本资料为23级人工智能相关的课程资料，由于其他课基本都与计科和数据重复且github已经有前辈上传了，于是此处不进行上传，希望发现此资料的学弟学妹们能够也将自己的帮助传给自己的学弟学妹，薪火相传。
### 此处为部分课的经验：
#### c语言（谢zh老师）：需要去做一些往年的真题课，知识掌握的很牢固如果没有去做些真题也容易被一些坑给拖分
#### c语言实验：同样需要多敲代码
#### 人工智能导论（徐y老师）：非水课，还是需要听一下，对后面后续课程的学习有帮助，考的和往年卷子部分重复，不可死记硬背
#### 概率论（刘xm老师）：考的几乎全是往年卷子真题
#### 计组（只有一个老师）：有往年真题可以参考，考的和画的重点几乎重合
#### 数据库(叶r老师）：b站有个课可以看，看完就差不多了
#### 数据结构（伍yl老师）：上课小测要注意，算平时分且占比大，考的不难
#### 计算机网络（青s老师）：考的难，考的可以参考他给的课后题目，有部分重复
#### 操作系统（jhy老师）：考的还行，出的题几乎全是考研真题。
#### 矩阵计算（伍xh老师）:考前我发现伍老师是东大毕业，所以考的和东大的差不多，我提供的资料也为东大资料，和期末考有部分重叠
#### 机器学习（舒x老师）：这门课需要认真学习，考的和往年部分重复，但是更多靠自己的掌握，有些上课没教
#### 神经网络（翟qy老师）：这门课和机器学习差不多，但是难一点，还有代码题，资料内卷子为往年真题，重复率很大
#### 计算机视觉（黄jx老师）：黄老师硕士似乎在cmu读的，所以用的也是cmu的资料，教的较难但是考的简单，平时分占70
#### 最优化（罗y老师）：上的简单考的也简单，老师最后会画重点，注意：要了解算法公式 
#### 软工（两个老师）：这门课我们这届画重点和出题不是一个老师，直接gg，上课能听一听还是听听吧~
#### 计算机图形学（谢zh老师）：水，小开，速成即可满绩
### 此处为给大一想要做科研的学弟学妹一些小建议：
学弟学妹们可以不急着去从众做科研，可以先学一些先修课，例如cs224等，学完后大二对老师们有大致了解再去做科研我觉得也来得及，切勿有从众心理，适合自己的才是最好的，想好自己以后想要做什么。
